### Para Linux

1: [Baixe o Python](https://www.python.org/downloads/)

2: [Baixe a última versão da SexAI](https://github.com/Virniciu/SexAI/releases/latest)

3: Abra um terminal dentro da pasta da SexAI e rode o seguinte comando:
```
python sexai.py
```
